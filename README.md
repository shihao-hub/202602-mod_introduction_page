# 能力勋章介绍页

饥荒联机版《能力勋章》模组的官方介绍页面，提供最新最全的勋章信息展示。

## 功能特性

- **勋章展示** - 卡片式展示所有勋章的详细信息
- **搜索功能** - 支持按勋章名称和代码搜索
- **主题筛选** - 按烹饪、战斗、生存、魔法、建造等主题分类
- **响应式设计** - 支持桌面端和移动端访问
- **深色主题** - 精美的深色界面设计
- **交互动画** - 流畅的卡片悬停和页面切换动画

## 页面结构

- **道具介绍** - 展示所有勋章的详细信息
- **模组说明** - 模组功能介绍
- **常见问题** - FAQ 常见问题解答
- **更新日志** - 模组版本更新记录
- **作者留言** - 作者寄语

## 使用方法

直接在浏览器中打开 `index.html` 文件即可查看，无需额外配置。

## 技术栈

- 纯 HTML/CSS/JavaScript
- 响应式 CSS Grid 布局
- CSS 变量主题系统
